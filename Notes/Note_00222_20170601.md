### 详细内容

#### 启动 Apache

终端输入

```shell
$ sudo apachectl start
```

可启动 Apache，默认站点的根目录为系统级根目录 

```
/Library/WebServer/Documents
```

#### 更多信息，可参考 [简书 - Mac OS X 启用 Web 服务器](http://www.jianshu.com/p/d006a34a343f)(推荐)

* 创建用户级根目录
* 启动 PHP
* 安装 MySQL
* 开启 HTTPS
